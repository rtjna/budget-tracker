# Budget Tracker

A personal, self-hosted spending tracker with automatic Monzo sync, CSV import
for other UK banks, and a categorization engine that learns from corrections.

Single-user, non-commercial. See [ROADMAP.md](ROADMAP.md) for the plan,
[PRIVACY.md](PRIVACY.md) for the privacy notice, and [TERMS.md](TERMS.md)
for terms of use.
